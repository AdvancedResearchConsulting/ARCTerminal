
Released for OSX
Works on Sierra v.10.12.6
You'll want to have an ARC product, or an arduino handy for testing. 
.......................
  Installation
unzip the application file

  Operation
right-click and select "open"
once the application runs, select the left-hand "gear" icon
configure for your device
click OK
click the "connect" icon (top left side) 
you should now be connected to the device, and able to use the buttons or enter text into the terminal window.
